## 🌟 **Phần thuyết phục khách hàng**

### 🌿 **Nguồn gốc & Chất lượng**
Section đẹp với gradient background, gồm 3 cards:

1. **🌿 Nguồn gốc**
   - Vườn nhãn tại Hưng Yên & Bắc Giang
   - Vùng đất vàng trồng nhãn
   
2. **🌾 Thu hoạch**
   - Thu hoạch buổi sáng sớm
   - Đảm bảo độ tươi và ngọt tự nhiên
   
3. **📦 Đóng gói**
   - Thùng carton chống sốc
   - Lót giấy mềm bảo vệ quả

### 📝 **Mô tả chi tiết**
- Mô tả đầy đủ về sản phẩm
- 4 điểm cam kết chính với icon check màu xanh:
  - Thu hoạch trong ngày
  - Không chất bảo quản
  - Đóng gói cẩn thận
  - Nguồn gốc rõ ràng

### 📦 **Hướng dẫn bảo quản**
Background màu kem #FFF8E7, riêng biệt cho:
- **Nhãn tươi**: Tủ lạnh 5-10°C, dùng trong 5-7 ngày
- **Nhãn sấy**: Nơi khô ráo, tránh ánh nắng, dùng trong 2-3 tuần

Tips chi tiết:
- Nhãn tươi: Không rửa trước khi bảo quản, để riêng, dùng túi kín
- Nhãn sấy: Tránh ẩm mốc, đậy kín, có thể bảo quản tủ lạnh

### 🚚 **Chính sách giao hàng & đổi trả**

**Giao hàng:**
- ✅ Miễn phí từ 499k

**Đổi trả:**

## 🎨 **Thiết kế theo rule.md**

### Màu sắc
- Primary Green: `#43A047` - Nút, active state
- Dark Green: `#1F5E3B` - Headings
- Yellow: `#F4B942` - Badges, ratings
- Red: `#DC2626` - Giá sản phẩm
- Light Green: `#DFF5E1` - Background sections
- Cream: `#FFF8E7` - Background sections

### Typography
- Heading: `text-4xl` (36-48px)
- Section heading: `text-2xl` (24px)
- Body: `text-base` (16px)
- Small: `text-sm` (14px)
- Tiny: `text-xs` (12px)

### Border Radius
- Cards: `rounded-3xl` (24px)
- Buttons: `rounded-full`
- Small elements: `rounded-2xl` (20px)

## 🔧 **Cách sử dụng**

### Bước 1: Click vào sản phẩm
Trên trang chủ, click vào:
- Ảnh sản phẩm
- Tên sản phẩm

→ Tự động chuyển đến trang chi tiết

### Bước 2: Xem thông tin
- Xem ảnh gallery (click thumbnail để đổi ảnh)
- Đọc mô tả chi tiết
- Xem nguồn gốc, cách bảo quản

### Bước 3: Chọn sản phẩm
1. Chọn biến thể (1kg, 2kg, 500g...)
2. Chọn số lượng (nút +/-)
3. Click "Thêm vào giỏ hàng"

### Bước 4: Quay lại hoặc xem sản phẩm khác
- Nút "Quay lại trang chủ"
- Click vào sản phẩm liên quan

## 📊 **Data Structure**

Mỗi sản phẩm cần có:

```typescript
{
  id: string;
  name: string;
  description: string;
  price: number;
  oldPrice?: number;          // Giá cũ (optional)
  unit: string;               // 'kg', 'túi', 'hộp', 'chùm'
  image: string;              // Ảnh thumbnail
  images: string[];           // 4 ảnh cho gallery
  category: string;           // 'Nhãn lồng tươi' hoặc 'Nhãn sấy'
  fullDescription: string;    // Mô tả đầy đủ
  origin: string;             // Nguồn gốc
  harvest: string;            // Cách thu hoạch
  packaging: string;          // Cách đóng gói
  badge?: string;             // 'Bán chạy', 'Mới', 'Quà tặng'
  rating: number;             // 1-5
  variants: [                 // Các biến thể
    {
      id: string;
      name: string;           // '1kg', '2kg', '500g'
      price: number;
      oldPrice?: number;
      stock: number;
    }
  ]
}
```
